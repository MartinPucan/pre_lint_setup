# ESLint, Prettier, Husky git hooks setup

## Installation

### Husky
> npm install husky --save-dev

> npx husky install

> npm pkg set scripts.prepare="precommit"

### Lint staged
> npm install --save-dev lint-staged

### Prettier
> npm install --save-dev prettier

### ESlint
> npm init @eslint/config
